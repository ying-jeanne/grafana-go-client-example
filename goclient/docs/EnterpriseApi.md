# **AddBuiltinRole**
> SuccessResponseBodyModel AddBuiltinRole(ctx, body)
Create a built-in role assignment.

You need to have a permission with action `roles.builtin:add` and scope `permissions:type:delegate`. `permissions:type:delegate` scope ensures that users can only create built-in role assignments with the roles which have same, or a subset of permissions which the user has. For example, if a user does not have required permissions for creating users, they wonâ€™t be able to create a built-in role assignment which will allow to do that. This is done to prevent escalation of privileges.
